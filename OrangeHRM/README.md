# QA Automation Assignment

## Application Test:
App Name: OrangeHRM

URL: https://opensource-demo.orangehrmlive.com/

OrangeHRM application is finance-related HR and payroll workflows, making it suitable for testing.

## Framework & Language
- Language: Java
- UI Automation tool: Selenium WebDriver
- Test Runner: TestNG
- Build Tool: Maven
- IDE: Eclipse

##Run Testng file

Run in Eclipse
Import as Existing Maven Project

Right-click on testng.xml → Run As → TestNG Suite

### Prerequisites
1. Java 11+ installed
2. Maven installed
3. Chrome browser installed
4. ChromeDriver (managed automatically by WebDriverManager)
